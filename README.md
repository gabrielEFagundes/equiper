# Equiper

## Project Overview
Equiper is a React-based application that serves as an equip sorter, utilizing the randomuser.me API to offer a seamless user experience in displaying random users.

## Features
- **User Display**: Showcasing random users in visually appealing cards.
- **Filtering**: Allows users to filter the displayed cards by gender.
- **Senior Identification**: Users above the age of 50 are marked as seniors for easy identification.
- **Talent Tracking**: Keeps track of the total number of found talents, providing users with valuable insights.

## Technology Stack
- **React** for building the user interface.
- **TypeScript** for a robust coding experience and type safety.
- **Vite** as the build tool, ensuring a fast and efficient development environment.
